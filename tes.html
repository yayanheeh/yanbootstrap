Desain Bahan untuk Bootstrap

Desain Material untuk Bootstrap adalah tema untuk Bootstrap 3 yang memungkinkan Anda menggunakan yang baru Desain Bahan Google dalam kerangka front-end favorit Anda.

Jika Anda menyukai proyek ini, Anda dapat mendukung saya dengan menyumbangkan sesuatu di Gratipay, membintangi repositori ini, atau melaporkan bug dan ide .

Baca lebih lanjut tentang Desain Bahan untuk Bootstrap di halaman Github .

Anda juga dapat menginstal dan mengelola Bootstrap Material menggunakan Bower

Apa yang termasuk

Desain Material untuk Bootstrap dapat diunduh dalam dua bentuk. Pertama, karena lebih sedikit / sass file sumber. Kedua, saat mengompilasi file .css.

Kerangka yang Diperlukan

Bootstrap v3.0 +

Tema ini memperluas, gaya, dan memodifikasi elemen dan gaya Bootstrap. Tanpa Bootstrap proyek ini tidak akan ditampilkan dengan benar. Untuk menginstal Bootstrap, silakan kunjungi Dapatkan Bootstrap

jQuery 1.9.1+

Semua plugin JavaScript membutuhkan jQuery untuk disertakan.

Jika mengunduh alih-alih menggunakan Bower, setelah diunduh, unzip folder yang dikompresi untuk melihat struktur (yang dikompilasi) Desain Material untuk Bootstrap. Anda akan melihat sesuatu seperti ini: 

  Bahan /

                │ ├── bootstrap-material-design.css
                │ ├── bootstrap-material-design.css.map

                │ ├── bootstrap-material-design.min.css.map
                │ ├── ripples.css.map
                │ ├── riak.min.css

                ├── js /
                │ ├── material.js
                │ ├── material.min.js
                │ ├── material.min.js.map
                │ ├── riak.js
                │ ├── riak.min.js

Cukup salin file CSS dan JS yang dikompilasi dan file font dari .zip dan tambahkan ke situs Anda.


Setelah disalin, Anda perlu menginisialisasi javascript materi dengan menambahkan javascript berikut ke situs Anda, 

$.material.init()
